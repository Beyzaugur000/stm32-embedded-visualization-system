# STM32 OLED Distance Visualization

An embedded systems project developed using STM32 microcontrollers, OLED displays, and ultrasonic sensors.

The project measures distances using an ultrasonic sensor and visualizes the data on an OLED display in real time. It also includes graphical rendering algorithms and geometric shape drawing implementations such as the Turkish crescent-star symbol.

## Features

* Ultrasonic distance measurement
* OLED display visualization
* Real-time graphical rendering
* PWM signal processing
* STM32 timer interrupt handling
* Embedded systems programming
* Geometric drawing algorithms
* Turkish crescent-star symbol rendering

## Hardware Components

* STM32F407VGT6
* HC-SR04 Ultrasonic Sensor
* OLED Display
* Active Buzzer
* Breadboard
* Jumper Cables

## Technologies

* Embedded C
* STM32
* PWM
* Timer Interrupts
* OLED Display Programming
* Ultrasonic Sensors
